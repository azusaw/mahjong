# about
Mahjong score calculation tool for my lovely team A-HAN

# website
> https://a-han.tk/

# used
 - Nuxt.js
 - Vuetify
 - Netlify

# build setup
```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
